Формат файла для импорта:

Файл должен содержать вопросы в следующем формате:

Question: Текст вопроса
Options:
[ ] Вариант ответа 1
[ ] Вариант ответа 2
[*] Правильный вариант ответа
[ ] Вариант ответа 4
  • Каждый вопрос начинается с Question: и текста вопроса
  • За ним следует строка Options:
  • Затем перечисляются варианты ответов, где правильный отмечен звездочкой [*]
  • Неправильные варианты отмечены пустыми скобками [ ]
  • Между вопросами не должно быть дополнительных строк
